

## About Project

It's a web app developed using Laravel and Livewire for restaurant management.

- Menu Page
- Menu Modification Interface
- Diverse Administrative Panels
- Simplistic Design
- Statistics Dashboard
- User Management System
- Order Management Tools
- Checkout Process
